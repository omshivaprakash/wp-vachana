== Description ==
Vachana Sanchaya is a website dedicated for linquistic research on 11th and 12th Century literature called "Vachana Sahitya". Provides access to 21000 Vachana's (verses of poems) written by 258 Vachanakaara's (Poets/Creators of those Vachanas) in Unicode.  This widget is disgned to display the daily vachana from the site vachana.sanchaya.net to wordpress blogs. 

Visit <a href="http://vachana.sanchaya.net/">Vachana Sanchaya</a> for more information. Visit <a href="http://en.wikipedia.org/wiki/Vachana sahitya">Vachana Sahitya</a> page on Wikipedia for more information on Vachana Sahitya movement.

== Installation ==

1. Upload the `vachana` folder to the `/wp-content/plugins/` directory
2. Activate the Vachana Sanchaya plugin through the 'Plugins' menu in WordPress
3. Go to Appearance --> Widgets to add the Vachana Sanchaya Daily Vachana widget to any widgetized areas, such as sidebars.

== Frequently Asked Questions ==

= How to style the widget? =
The vachana output wrapped in `<p>` tag with `vachana` ID and class. Use `#vachana` or `.vachana` in your Custom CSS to style it.
